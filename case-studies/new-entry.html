Modernizing the Entry Creation Experience

A guided, intuitive workflow for creating structured entries with prefix logic, validation, and a clean two‑panel layout.

Overview

The New Entry workflow was redesigned to eliminate user confusion, reduce data errors, and modernize the UI. The previous version relied on outdated SharePoint forms, inconsistent spacing, and manual prefix entry. The new design introduces a structured, guided experience with real‑time validation and a modern layout.

The Problem

  • Users struggled with inconsistent prefix formatting.
  • Form fields lacked hierarchy and visual grouping.
  • Validation occurred only after submission, causing rework.
  • The UI felt outdated and did not match modern M365 patterns.

The Solution

I designed a clean, two‑panel interface with a left‑side creation form and a right‑side live preview. Prefix selection is now driven by a dropdown that automatically applies formatting rules. Validation is instant, and spacing follows a consistent 4/8/16px rhythm.

Key Features

  • Responsive Layout: Works across desktop and tablet.
