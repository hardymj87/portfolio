Interaction Patterns

Consistent hover, focus, and active states improve usability and accessibility.

  • Hover: Subtle elevation or color shift.
  • Focus: High‑contrast outline for keyboard accessibility.
  • Active: Slight press‑down effect or darker shade.
  • Error: Red border + inline message.
  • Success: Green confirmation message.
